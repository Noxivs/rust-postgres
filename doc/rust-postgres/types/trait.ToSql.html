A trait for types that can be converted into Postgres values

Required Methods

fn to_sql(&self, ty: &PostgresType) -> (Format, Option<~[u8]>)

Converts the value of self into a format appropriate for the Postgres backend.

Failure

Fails if this type cannot be converted into the specified Postgres type.
