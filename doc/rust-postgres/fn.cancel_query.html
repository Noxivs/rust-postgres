Function rust-postgres::cancel_query

pub fn cancel_query(url: &str, ssl: &SslMode, data: PostgresCancelData) -> Result<(), PostgresConnectError>

Attempts to cancel an in-progress query.

The backend provides no information about whether a cancellation attempt was successful or not. An error will only be returned if the driver was unable to connect to the database.

A PostgresCancelData object can be created via PostgresConnection::cancel_data. The object can cancel any query made on that connection.
